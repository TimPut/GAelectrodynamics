\makedefinition{Multivector Fourier transform pair.}{dfn:greensFunctionOverview:280}{
The \textit{Fourier transform pair} for a multivector function \( F(\Bx, t) \) will be written as
\begin{equation*}
\begin{aligned}
F(\Bx, t) &= \int F_\omega(\Bx) e^{j \omega t} d\omega \\
F_\omega(\Bx) &= \inv{2 \pi} \int F(\Bx, t) e^{-j \omega t} dt,
\end{aligned}
\end{equation*}
where \( j \) is an arbitrary scalar imaginary that commutes with all multivectors.
} % definition
